The column-gap property sets the gap between the columns:

.grid-container { display: grid; column-gap: 25px; }

The row-gap property sets the gap between the columns:

.grid-container { display: grid; row-gap: 25px; }

We use Grid lines to arrange grid items

.name_of_your_grid { grid-column-start: 1; grid-column-end: 3; }

.name_of_your_grid { grid-row-start: 1; grid-row-end: 3; }
