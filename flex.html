By using container with "flex-wrap: nowrap;" will never wrap its items.I repeat never

I have following properties

flex-direction: column; The column value stacks the flex items vertically (from top to bottom)

flex-direction: column-reverse; The column-reverse value stacks the flex items vertically (but from bottom to top)

flex-direction: row The row value stacks the flex items horizontally (from left to right)

flex-direction: wrapThe wrap value specifies that the flex items will wrap if necessary

flex-direction: nowrapThe nowrap value specifies that the flex items will not wrap (this is default)
